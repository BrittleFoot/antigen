# Oantigen


## Installation

```bash
python -m pip install -r requirements.txt
```

## Usage

```bash
# get gff with operons-specific features

python merge-operons.py a.gff list_of_operons orfs_coordinates -o b.gff

# draw operons
python draw-operons.py b.gff

```
